# DP;Dynamic Programming 동적계획법

하나의 큰 문제를 여러개의 작은 문제로 나누어 그 결과를 저장하고 다시 큰 문제를 해결한다.

일반적인 재귀 방식또한 DP와 유사하지만 가장 큰 차이점은 여러번 반복될 수 있는 작은 문제를 단 한번만 수행하게 한다는 점에 있음.

O(n^2) -> O(f(n)) 만큼 성능을 향상시킬 수 있음
